## Official card thumbnails can be found in [`https://sekai.best/card`](https://sekai.best/card).

1. **Right-click** on the card thumbnail and select "Inspect Element"
<img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/a45726c8-0615-4fb8-9abd-f8d732dba06e" />

2. Find the source link that starts with `storage.sekai.best` and click it
<img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/76a935e3-1260-4226-9b87-9f6d02f8ac66" />

3. Change the file extension in the URL field: `.webp` → `.png`
<img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/f5765c57-ceb2-4c77-812c-2ef14679716f" />

4. **Right-click** image and select "Save image as..."
<img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/4788aa93-0fbd-4248-b1ae-eca5b12d6e35" />

## 公式カードのサムネイルは、[`https://sekai.best/card`](https://sekai.best/card)にあります。

1. カードのサムネイルを**右クリック**し、「要素を検査」を選択します。
<img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/a45726c8-0615-4fb8-9abd-f8d732dba06e" />

2. `storage.sekai.best` で始まるソースリンクを探して、それをクリックしてください
<img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/76a935e3-1260-4226-9b87-9f6d02f8ac66" />

3. URL フィールドのファイル拡張子を`.webp`から`.png`に変更します。
<img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/f5765c57-ceb2-4c77-812c-2ef14679716f" />

4. 画像を**右クリック**し、「画像を別名で保存...」を選択します。
<img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/4788aa93-0fbd-4248-b1ae-eca5b12d6e35" />
